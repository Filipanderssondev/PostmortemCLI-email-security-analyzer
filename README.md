# Post-Mortem - email security analyzer
This is my final, graduation project in Chas Academy. This is "Post-Mortem", A containerized sandbox enviroment and analyzer for e-mails containing potentially malicious content or malware. This tools is developed soley by me, at the request of SMHI, as there a demand for these type of tools within municipal government organizations.
